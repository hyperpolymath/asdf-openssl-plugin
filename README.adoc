toc::[]

== Overview

=== What is OpenSSL?

OpenSSL is the de facto TLS/SSL toolkit and cryptography library for secure
networking, certificate management, and key operations. It is used across
servers, developer tooling, and build pipelines where cryptographic primitives
and TLS are required.

=== What is asdf?

{url-asdf}[asdf] is a universal version manager that allows you to manage
multiple runtime versions with a single CLI tool.

=== Why asdf-openssl?

[cols="1,3"]
|===
|Feature |Benefit

|**Version Switching**
|Switch between OpenSSL versions per project

|**Project Isolation**
|Pin OpenSSL per repository via `.tool-versions`

|**Reproducible Builds**
|Align CI/CD and developer environments to the same OpenSSL version

|**Cross-Platform**
|Works on Linux and macOS, with Windows via WSL
|===

=== Dependencies

[source,bash]
----
# Debian/Ubuntu
sudo apt-get update
sudo apt-get install -y curl git build-essential

# Fedora/RHEL
sudo dnf install -y curl git gcc

# macOS (via Homebrew)
brew install curl git
xcode-select --install
----

=== asdf Installation

[source,bash]
----
# Clone asdf
git clone https://github.com/asdf-vm/asdf.git ~/.asdf --branch v0.14.0

# Add to your shell (bash)
echo '. "$HOME/.asdf/asdf.sh"' >> ~/.bashrc
echo '. "$HOME/.asdf/completions/asdf.bash"' >> ~/.bashrc

# Reload your shell
exec $SHELL
----

== Installation

=== Adding the Plugin

[source,bash]
----
asdf plugin add openssl https://github.com/hyperpolymath/asdf-openssl-plugin.git
asdf plugin list
----

=== Installing OpenSSL Versions

[source,bash]
----
asdf list all openssl
asdf install openssl latest
asdf install openssl 3.3.1
----

=== Setting the Version

[source,bash]
----
asdf global openssl 3.3.1
asdf local openssl 3.3.1
asdf current openssl
openssl version
----

== Usage

=== Project Configuration

[source]
----
openssl 3.3.1
----

== Configuration

This plugin uses standard asdf behavior. No additional configuration is
required.

== Troubleshooting

* Ensure `openssl` is on PATH after installation.
* Remove stale builds by uninstalling and reinstalling the version.
* Verify your shell is sourcing the asdf scripts.
